# Basic EMS using Spring Boot and React

Employee Management System built using:

- Spring Boot
- React JS
- MySQL
- REST API

## Features

- Add Employee
- Update Employee
- Delete Employee
- View Employee List
- Backend API using Spring Boot
- Frontend using React

## Technologies Used

### Frontend
- React
- Axios
- Bootstrap

### Backend
- Spring Boot
- Spring Data JPA
- MySQL

## How to Run

### Backend

1. Open backend project
2. Configure MySQL in `application.properties`
3. Run Spring Boot application

### Frontend

```bash
npm install
npm start
```
